# projeto_rh_dados_de_recrutamento
Projeto de People Analytics: análise de eficiência de processos seletivos via Python. O estudo explora dados de contratação, avalia o ROI de fontes de talentos, compara KPIs por departamento e investiga tempos de fechamento de vagas, transformando métricas de RH em insights estratégicos para otimizar o funil de talentos.
