# Análise de Eficiência de Recrutamento (People Analytics)

Este projeto simula uma auditoria de dados de um processo seletivo, utilizando Python para identificar gargalos, fontes de talentos mais assertivas e comparar a performance entre departamentos.

### 🎯 Objetivo
Transformar dados brutos de contratação em insights estratégicos para a tomada de decisão no RH.

### 🛠 Tecnologias
- Linguagem: Python
- Bibliotecas: Pandas, Matplotlib

### 📊 Principais Insights
- **Eficiência de Fontes:** Identifiquei que LinkedIn possui a maior taxa de conversão para contratações.
- **Performance:** Departamentos como [Financeiro/RH] performam acima da média da empresa, enquanto [TI/Marketing] podem necessitar de revisão no processo de avaliação técnica.
- **Tempo de Fechamento:** O tempo médio de 24.6 dias foi utilizado como parâmetro para identificar desvios dos processos em cada departamento.

### 📂 Estrutura
- `Projeto_RH_Dados_de_Recrutamento.ipynb`: Notebook com a análise completa, tratamento de dados e visualizações.
- `dados_recrutamento_100.csv`: Base de dados sintética utilizada para o exercício.
- `Criando_base_de_Dados.ipynb`: Script utilizado para gerar o dataset sintético de 100 registros.
